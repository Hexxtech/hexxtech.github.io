How to use

1. Enter a CORS proxy URL above (see note) if the toggle is ON
2. Paste an AFH link like https://androidfilehost.com/?fid=123456
3. Set parallel connections (more = faster but heavier)
4. Toggle Query Loop to retry until found
5. Hit Find Mirror — download starts automatically
